# **App Name**: Gastronomic Gateway

## Core Features:

- Restaurant Details Display: Showcase restaurant name, logo, contact information, and business hours.
- Offer Carousels: Display special offers, today's specials, and new items in horizontally scrollable carousels.
- Categorized Menu: Organize dishes into categories (e.g., starters, mains, desserts) with filtering options.
- Admin Authentication: Secure admin dashboard with Firebase Authentication to manage website content.
- Restaurant Details Management: Enable admin to edit restaurant information like name, logo, contact details, and opening hours.
- Offer Management: Allow admin to create, update, and delete offers and specials, including setting display priority.
- Menu Management: Admin interface for managing dish categories and individual dish items, including details, pricing, and availability.

## Style Guidelines:

- Primary color: Earthy Umber (#826247) for a warm and inviting feel.
- Background color: Creamy Beige (#F5F5DC), offering a soft, neutral backdrop.
- Accent color: Muted Gold (#B8860B) used sparingly to highlight key elements and calls to action.
- Body font: 'PT Sans', a humanist sans-serif suitable for headlines or body text.
- Mobile-first, responsive design with a clean, uncluttered layout to enhance user experience.
- Simple, elegant icons to represent menu categories and other key features.
- Subtle transitions and animations to provide feedback and guide the user through the app.
